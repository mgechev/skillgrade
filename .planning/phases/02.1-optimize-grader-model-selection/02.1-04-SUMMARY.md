## Decisions Made
- **Ollama env vars at job level:** Set on the test-integration job so they propagate to all steps, including when Ollama is started within tests.
- **Best-effort config warning:** The warning checks `process.env` in the Node.js process, not the Ollama server's actual configuration. This means false warnings are possible when Ollama was started separately (e.g., on Windows desktop). Accepted per user decision: "warns, not fails."
- **Once-per-instance warning:** The `warnedAboutConfig` flag prevents repeated warnings across multiple `grade()` calls on the same LLMGrader instance.
- **Thread count per platform:** CI uses OLLAMA_NUM_THREAD=4 (matching ubuntu-24.04-arm's 4 vCPU), README documents 12 for local Snapdragon X Elite.
